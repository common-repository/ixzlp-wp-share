== Description ==
ixzlp WP Share is a WordPress Plugin that displays social media sharing buttons under blog posts.

If you like ixzlp WP Share feel free to follow me on Twitter:
https://www.twitter.com/ixzlp

== Installation ==
1. Upload the plugin files to the "wp-content/plugins/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Use the "Settings -> ixzlp WP Share" screen to configure the plugin.
